## 使用

>

1. **安装**
```shell

```

2. 命令

```shell

free create projectName 
  - -y
  - -v
```

3. **自由配置模板选项和功能**

> 用户路径下会有一个 `.free-cli` 文件夹。

> `template` 缓存远端的模板，加速创建时间
